# IT League

Landing page for the IT League project.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- npm

## Growth Path

- Admin panel: protected Next.js routes with role-based access.
- Data layer: Prisma with PostgreSQL when judges, FAQ blocks, applications, and payments need persistence.
- Payments: WayForPay integration through server-side Next.js route handlers.
- Content: database-backed sections first, then optional CMS if non-technical editing becomes a priority.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```
